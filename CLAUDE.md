# プロジェクト概要

このプロジェクトは**t106.net**の個人Webサイトで、動的なビジュアルエフェクトを特徴とする創造的なポートフォリオサイトです。

## 基本情報

**メインサイト**: [t106.net](https://t106.net)  
**作者**: 106（Python/機械学習/Linuxを専門とする）

## 主要機能

1. **動的ビジュアルシステム** - 4種類の異なるアニメーション効果をランダムまたは指定で表示
   - **Boltzmann Machine** (`bm`): 物理学ベースのセルオートマトン
   - **Gaussian Process** (`gp`): 統計的データ可視化
   - **Rain** (`rain`): リアルな水面波紋シミュレーション  
   - **Rez** (`rez`): 3D幾何学アニメーション

2. **プロフィール表示** - 中央にt106.netブランディングとソーシャルリンク（Twitter、GitHub、SlideShare、Steam）

3. **柔軟な表示制御** - URLクエリパラメータで特定モードの選択やプロフィール非表示が可能

## 技術スタック

- **フロントエンド**: HTML5、CSS3、JavaScript（ES6+）
- **3Dグラフィックス**: Three.js、WebGL
- **2Dグラフィックス**: PIXI.js、Canvas API  
- **機械学習**: TensorFlow.js
- **UI**: Bootstrap 4、Font Awesome
- **解析**: Google Analytics

## ビジュアルエフェクト詳細

### js/alter.js - スクリプトマネージャー
- 4つの異なるビジュアルモードを管理する中央制御システム
- URLクエリパラメータまたはランダム選択でモード決定
- 外部依存関係とモジュールスクリプトの動的読み込み

### js/bm/ - Boltzmann Machine
- イジングモデルベースの物理シミュレーション
- ギブスサンプリングアルゴリズムによる確率的状態更新
- 温度制御可能なセルラーオートマトン

### js/gp/ - Gaussian Process可視化
- PIXI.jsによる高性能2Dレンダリング
- TensorFlow.jsを使用した数学的計算
- 信頼区間と散布図を含む統計データ可視化

### js/rain/ - 水面波紋シミュレーション
- 波動方程式による現実的な水面アニメーション
- ポアソン分布による自然な水滴生成
- 円形波の伝播と動的色マッピング

### js/rez/ - 3D幾何学アニメーション
- Three.jsによるWebGL 3Dレンダリング
- 有機的な変形を伴う複数のリング構造
- MeshLineによる滑らかなワイヤーフレーム表現

現代的なWebGL技術と数学的アルゴリズムを組み合わせた、技術デモンストレーション兼個人ブランディングサイトです。
